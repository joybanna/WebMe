De'sir The fragment of Memory (Thesis)
Desir is a 2D Semi idle Heroes collection Game. the player can collect and use Heroes to play game
The game has many systems.
Characters collection and character stats system
Inventory and items system
Crafting and selling system items
Dialogs system
Battle system is 2 modes auto and manual. Auto mode player must set action before the battle. Manual mode player uses action in the battle.
